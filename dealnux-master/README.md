# Dealnux

Dealnux is a modern multi-vendor commerce platform built with Expo, React Native, and TypeScript.

The platform combines:
- Multi-vendor marketplace
- Third-party product aggregation
- Real-time price drop tracking
- AI-powered chatbot
- Push notifications
- Cross-platform mobile commerce experience

---

## Tech Stack

- Expo
- React Native
- TypeScript
- React Query
- Zustand / Redux Toolkit
- Node.js
- MongoDB / PostgreSQL
- Firebase Cloud Messaging

---

## Features

- Seller product management
- External marketplace integration
- Real-time price monitoring
- Push notification alerts
- AI commerce assistant
- Scalable mobile architecture

---

## Getting Started

```bash
npm install
npx expo start
```
